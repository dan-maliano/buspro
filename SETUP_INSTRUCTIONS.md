# הוראות הגדרה - ייבוא 500 שאלות

## שלב 1: הרצת סקריפט הייבוא

הרץ את הפקודה הבאה בטרמינל:

\`\`\`bash
npm run seed
\`\`\`

הסקריפט יבצע את הפעולות הבאות:
1. ימחק את כל השאלות הישנות מהדטאבייס
2. ייבא את כל 500 השאלות מקובצי ה-JSON
3. יוודא שהתשובות הנכונות מוגדרות כראוי
4. יציג דוח מפורט על החלוקה לפי פרקים

## שלב 2: אימות

לאחר הרצת הסקריפט, וודא שהדוח מציג:
- ✓ 500 שאלות בסך הכל בדטאבייס
- ✓ חלוקה נכונה לפי 13 פרקים
- ✓ כל פרק מכיל את מספר השאלות הצפוי

## שלב 3: בדיקת המבחן

1. פתח את האתר
2. התחל מבחן סימולציה חדש
3. וודא:
   - ✓ יש בדיוק 30 שאלות במבחן
   - ✓ השאלות מחולקות לפי הפרקים (6 מפרק 1, 2 מפרק 2, וכו')
   - ✓ התשובות מחושבות נכון
   - ✓ עד 8 טעויות = עובר, 9+ טעויות = נכשל

## פתרון בעיות

אם הסקריפט נכשל:
1. וודא ש-`NEXT_PUBLIC_SUPABASE_URL` מוגדר
2. וודא ש-`SUPABASE_SERVICE_ROLE_KEY` מוגדר
3. בדוק שקבצי data/questions.json ו-data/correct_answers.json קיימים
4. בדוק את ה-console logs לפרטים נוספים
